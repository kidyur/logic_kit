# Logic Expressions Kit - console app

Приложение для обработки булевых выражений.
1) Упрощение (приведение к базису из трех операторов: И, ИЛИ, НЕ)
2) Приведение к ДНФ (Дизъюнктивной Нормальной Форме)
3) Приведение к КНФ (Конъюнктивной Нормальной Форме)

| Оператор | Операция | Оператор | Операция  |
| -------- | -------- | -------- | --------- |
| &        | AND      | >        | IMPLIFIES |
| \|       | OR       | !        | NOR       |
| ~        | NOT      | /        | NAND      |
| +        | XOR      | =        | EQUALS    |

Операнды могут быть только буквами английского алфавита в нижнем регистре.
Если между операндами нет оператора, между ними ставится & (AND):
ab = a & b
